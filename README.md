# Aevra
Aevra is an AI-powered life companion that transforms memories into cinematic stories, intelligent insights, voice journals, and personalized reflections—all in one beautifully designed platform.

A calm, private journal app with auth, entries, media, graph views, and insights. The backend is a FastAPI app that runs locally and the frontend is a Next.js app.

## Run everything

From the project root in PowerShell:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\start.ps1
```
## Requirements

- Python 3.11+
- Node.js 18+
- Optional: Ollama for reflection generation
